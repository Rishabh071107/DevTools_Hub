# 🚀 DevTools Hub

> A modern all-in-one developer toolkit focused on JSON processing and transformation.

DevTools Hub is a fast, browser-based utility designed to help developers repair, validate, format, compare, and convert JSON data through a clean, professional workspace inspired by VS Code and Postman.

---

## ✨ Features

### 🔧 Smart JSON Repair
Automatically repairs common JSON syntax errors such as:

- Missing quotes
- Missing commas
- Missing braces
- Missing brackets
- Missing colons
- Single quotes
- Trailing commas
- Duplicate commas
- Unclosed arrays and objects

Also provides:

- Repair summary
- JSON statistics
- Copy & download options

---

### 📝 JSON Formatter

- Pretty Print
- Minify
- Custom indentation
- Instant formatting

---

### ✅ JSON Validator

- Validate JSON
- Detailed error messages
- Line and column reporting

---

### 🔍 JSON Compare

Compare two JSON documents and highlight:

- Added values
- Removed values
- Modified values

---

### 🔄 JSON Converter

Supports conversion between:

- JSON ⇄ XML
- JSON ⇄ YAML
- JSON ⇄ CSV

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/devtools-hub.git
```

### Navigate to the Project

```bash
cd devtools-hub
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
